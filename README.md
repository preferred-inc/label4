# Label4

**伝票PDFを最大4枚、A4用紙1枚にまとめて印刷できる無料ウェブアプリ**

https://preferred-inc.github.io/label4/

## Features

- 2面付け / 4面付けレイアウト切り替え
- 複数ページPDF自動展開
- ページ追加で何枚でもA4シートを一括印刷
- ドラッグ&ドロップ対応
- PWA対応（ホーム画面追加可）
- データはブラウザ内のみで処理（サーバ送信なし）

## Supported Labels

- クリックポスト（日本郵便）
- レターパックライト / プラス
- ゆうパケット / ゆうパケットポスト
- スマートレター
- ネコポス（ヤマト運輸）
- その他PDF形式の伝票全般

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- PDF.js for PDF rendering
- GitHub Pages hosting
- PWA with Workbox
